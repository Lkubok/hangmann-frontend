# Hangmann

## Clone app into your directory

```
git clone git@github.com:Lkubok/hangmann-game.git
```

## Install it

```
npm install
```

## Running app:
```
npm run start
```

## Used technologies

### React.js, Redux.js, Formik, Sass, HighCharts.js
